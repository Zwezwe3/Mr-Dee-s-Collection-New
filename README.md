# Mr Dee's Collection

Simple product showcase website.

## Setup
- Upload all files to your GitHub repository root.
- Optionally add product images inside an `images/` folder and update image `src` paths.
- Enable GitHub Pages from the repository `Settings -> Pages -> Source: main branch (root)`.
